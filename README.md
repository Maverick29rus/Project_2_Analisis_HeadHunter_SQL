# Project 2 — Анализ резюме из HeadHunter (SQL)

![HeadHunter](https://raw.githubusercontent.com/AndreyRysistov/DatasetsForPandas/main/hh%20label.jpg)

## Описание задачи

Проект — учебное исследование рынка вакансий и требований к кандидатам на основе данных HeadHunter. Основная цель — продемонстрировать практические навыки в извлечении данных, написании сложных SQL-запросов, предобработке и исследовательском анализе данных с визуализацией результатов.

Проект включает в себя несколько этапов:

1. Знакомство с данными
2. Предварительный анализ (EDA)
3. Детальный анализ вакансий
4. Анализ работодателей
5. Предметный анализ и формулирование выводов
6. Дополнительные исследования и сегментация

Ключевые навыки и технологии

- SQL: сложные запросы, агрегирование, использование COALESCE/CASE.
- Python и Jupyter Notebook: скрипты для предобработки и EDA.
- pandas: очистка данных, агрегации, преобразования.
- matplotlib: визуализация распределений и трендов.
- Аналитический подход: сегментация по опыту, индустрии и регионам; корректная агрегация зарплат.
